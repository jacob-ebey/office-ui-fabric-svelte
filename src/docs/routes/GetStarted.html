Get started with Fabric Svelte

Use NPM to get Fabric Svelte components and core styling.


1

To install the Fabric Svelte NPM package, from the root of your project, run:

npm install -s office-ui-fabric-svelte
2

The library includes commonjs (dist/index.common.js), es modules (dist/index.js) and UMD (dist/index.umd.js). The raw Svelte components are also provided if you wish to compile it as part of your build process.
